About

鵜飼 里奈

Rina Ugai

東京都在住/京都生まれ鹿児島育ち/インドネシアと日本のクォーター/ 2児の母

世界中のどこにいても、誰かの役に立てる専門スキルを身に付けたいと思いデザインの勉強を始めました。
デザインは「課題を解決する為にある」と考えています。ただ綺麗でかっこいい（可愛い）デザインではなく、お客さまの思いや目的、課題を理解し、意味のあるデザインをいたします。

「デザインを通して少しでもお客様の役に立ちたい」そんな想いで活動しています。
ご相談やご依頼など、お気軽にご連絡ください。

【使用ソフト】 Adobe Illustrator / Adobe Photoshop / figma / Visual studio code
